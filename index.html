Estrutura Básica do HTML

[voltar]

A estrutura essencial de um documento HTML é:

< html >
< head > Informações carregadas antes do restante da página.< /head >
< body > Conteúdo que aparecerá no navegador. < /body >
< /html >

Falando sobre Tags

[voltar]

Estrutura básica das tags: < NOME-DA-TAG > CONTEÚDO A SER ENGLOBADO PELA TAG < / NOME-DA-TAG >.

Algumas tags não precisam de fechamento, como é o caso da tag INPUT (cria campo de inserção de dado pelo usuário) e da tag IMG (insere imagem na página ).

Atributos das Tags

[voltar]

Existem atributos gerais, que servem para a maioria das tags, e atribitos específicos para cada tag.

Exemplo: id, style, class (css) são atributos de vários tipos de elemento, enquanto src é específico do elemento img e href é atributo típico para links.

Lista completa de atributos das tags HTML no site da W3schools.

Textos

[voltar]

Há muitas tags 2 de formatação de texto. Entre as mais usadas, temos as de título (h1, h2, h3... até h6), parágrafo p, citação blockquote, sobrescrito sup, subscrito sub etc.

Algumas são especialmente importantes para a acessibilidade, como é o caso da tag em(ênfase), que embora visualmente tenha o mesmo efeito que a tag i (itálico), resulta em uma ênfase na voz emitida por leitores de páginas WEB utilizados, por exemplo, por pessoas com deficiência visual.

Lista completa de tags de formatação de texto no site da W3schools.

Links

[voltar]

O "a" vem de "âncora".

Atributo target:

  • retrocompatibilidade: frames, top, parentfuncionalidades em desuso
  • self: padrão
  • blank: outra aba

Links internos à página:

  • atributo id ao elemento a ser linkado;
  • atributo href="#id" no elemento a
